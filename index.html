Simplicity
Fract code blocks are not based on indentation, don't get in trouble with weird indentation errors, and not cause no-nonsense problems for beginners.

With clean and readable code blocks, it's easy to understand where each code block ends and the syntax is pretty simple.
  • No mandatory indention
  • Clean code blocks
  • Clean syntax
    • Performance
    Lexer works pretty fast. Before the code is run, it is parsed by the lexer and never goes through the parsing phase again.

    Interpreted languages ​​are inherently slower than compiled languages, but Fract is one of the fastest interpreted languages. It has a fast interpreter and implements a debugging algorithm as simple as possible.
  • No unnecessarry debugging
  • No retokenize
  • No conscious delay on interpreter processes
    • Safety
    Fract does not support null, it is impossible to get a null error.

    All definitions must be fully defined at the time of declaration and there is no default value assignment, the value must be given by the developer.
  • No null
  • No undefined values
  • No variable shadowing in blocks
  • No evaluate function
  • Immutability by default
    • Efficiency
    It has high memory efficiency and a powerful backend.

    Memory consumption is low, does not overload. It is a lightweight alternative for most systems with a light memory consumption.

    Memory Management
    Fract has garbage collector and memory management is done automatically. But if a developer wants to do it, why not?

    With the delete (del) keyword you can delete the definitions from the definition memory in the backend and that means free space in the memory eventually.
    var ( name = input("Name: ") surname = input("Surname: ") ) print("Hello " + name + " " + surname + "!") del name, surname
    Debugging
    Unreachable parts of code are never included in debugging.

    For example, the body of an if block, the body of a loop that is never executed, the body of a function that is never called... These are not included in debugging.
    func a() Unreachable code. end
