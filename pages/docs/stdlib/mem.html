func Malloc(size)
Malloc returns n size allocated list by specified size. Elements are actually none but is not literally none. For this reason, this is dangerous function if you not set each element with real value manually.
Special case is:
  • Malloc(size) = None if size is not integer


    • func Calloc(size, value=none)
    Calloc returns n size allocated and list, setted each elements sets as specified value.
    Special case is:
  • Calloc(size) = None if size is not integer


    • func Drop(name)
    Drop deletes define from current package source by specified name. Reports process state as boolean.
    Special cases are:
  • Drop(name) = error if "name" is not string
  • Drop(name) = false if "name" is empty


    • func Copy(*dest, const var src)
    Copy copies source list into destination list. And reports success state as boolean.
    Special case is:
  • Copy(dest, src) = false if "dest" or "src" is not list


    • func Defs()
    Defs returns all defines as list of current package.

    func GC()
    GC runs a garbage collection of Go and blocks the caller until the garbage collection is complete. It may also block the entire program.

    func IsReal(value)
    IsReal reports specified value is real or not.
