    func Abs(x)
    Abs returns the absolute value of x.
    Special cases are:
  • Abs(±Inf) = +Inf
  • Abs(NaN) = NaN


    • func Asin(x)
    Asin returns the arcsine, in radians, of x.
    Special cases are:
  • Asin(±0) = ±0
  • Asin(x) = NaN if x < -1 or x > 1


    • func Asinh(x)
    Asinh returns the inverse hyperbolic sine of x.
    Special cases are:
  • Asinh(±0) = ±0
  • Asinh(±Inf) = ±Inf
  • Asinh(NaN) = NaN


    • func Acos(x)
    Acos returns the arccosine, in radians, of x.
    Special cases are:
  • Acos(x) = NaN if x < -1 or x > 1


    • func Acosh(x)
    Acosh returns the inverse hyperbolic cosine of x.
    Special cases are:
  • Acosh(+Inf) = +Inf
  • Acosh(x) = NaN if x < 1
  • Acosh(NaN) = NaN


    • func Atan(x)
    Atan returns the arctangent, in radians, of x.
    Special cases are:
  • Atan(±0) = ±0
  • Atan(±Inf) = ±Pi/2


    • func Atan2(y, x)
    Atan2 returns the arc tangent of y/x, using the signs of the two to determine the quadrant of the return value.
    Special cases are:
  • Atan2(y, NaN) = NaN
  • Atan2(NaN, x) = NaN
  • Atan2(+0, x>=0) = +0
  • Atan2(-0, x>=0) = -0
  • Atan2(+0, x<=-0) = +Pi
  • Atan2(-0, x<=-0) = -Pi
  • Atan2(y>0, 0) = +Pi/2
  • Atan2(y<0, 0) = -Pi/2
  • Atan2(+Inf, +Inf) = +Pi/4
  • Atan2(-Inf, +Inf) = -Pi/4
  • Atan2(+Inf, -Inf) = 3Pi/4
  • Atan2(-Inf, -Inf) = -3Pi/4
  • Atan2(y, +Inf) = 0
  • Atan2(y>0, -Inf) = +Pi
  • Atan2(y<0, -Inf) = -Pi
  • Atan2(+Inf, x) = +Pi/2
  • Atan2(-Inf, x) = -Pi/2


    • func Atanh(x)
    Atanh returns the inverse hyperbolic tangent of x.
    Special cases are:
  • Atanh(1) = +Inf
  • Atanh(±0) = ±0
  • Atanh(-1) = -Inf
  • Atanh(x) = NaN if x < -1 or x > 1
  • Atanh(NaN) = NaN


    • func NaN()
    NaN returns an IEEE 754 ``not-a-number'' value.

    func Inf(sign)
    Inf returns positive infinity if sign >= 0, negative infinity if sign < 0.

    func IsNaN(x)
    IsNaN reports whether f is an IEEE 754 ``not-a-number'' value.

    func IsInf(x, sign)
    IsInf reports whether f is an infinity, according to sign.
    If sign > 0, IsInf reports whether f is positive infinity.
    If sign < 0, IsInf reports whether f is negative infinity.
    If sign == 0, IsInf reports whether f is either infinity.

    func Cbrt(x)
    Cbrt returns the cube root of x.
    Special cases are:
  • Cbrt(±0) = ±0
  • Cbrt(±Inf) = ±Inf
  • Cbrt(NaN) = NaN


    • func Dim(x, y)
    Dim returns the maximum of x-y or 0.
    Special cases are:
  • Dim(+Inf, +Inf) = NaN
  • Dim(-Inf, -Inf) = NaN
  • Dim(x, NaN) = Dim(NaN, x) = NaN


    • func Max(x, y)
    Max returns the larger of x or y.
    Special cases are:
  • Max(x, +Inf) = Max(+Inf, x) = +Inf
  • Max(x, NaN) = Max(NaN, x) = NaN
  • Max(+0, ±0) = Max(±0, +0) = +0
  • Max(-0, -0) = -0


    • func Min(x, y)
    Min returns the smaller of x or y.
    Special cases are:
  • Min(x, -Inf) = Min(-Inf, x) = -Inf
  • Min(x, NaN) = Min(NaN, x) = NaN
  • Min(-0, ±0) = Min(±0, -0) = -0


    • func Erf(x)
    Erf returns the error function of x.
    Special cases are:
  • Erf(+Inf) = 1
  • Erf(-Inf) = -1
  • Erf(NaN) = NaN


    • func Erfc(x)
    Erfc returns the complementary error function of x.
    Special cases are:
  • Erfc(+Inf) = 0
  • Erfc(-Inf) = 2
  • Erfc(NaN) = NaN


    • func Erfinv(x)
    Erfinv returns the inverse error function of x.
    Special cases are:
  • Erfinv(1) = +Inf
  • Erfinv(-1) = -Inf
  • Erfinv(x) = NaN if x < -1 or x > 1
  • Erfinv(NaN) = NaN


    • func Erfcinv(x)
    Erfcinv returns the inverse of Erfc(x).
    Special cases are:
  • Erfcinv(0) = +Inf
  • Erfcinv(2) = -Inf
  • Erfcinv(x) = NaN if x < 0 or x > 2
  • Erfcinv(NaN) = NaN


    • func Exp(x)
    Exp returns e**x, the base-e exponential of x.
    Special cases are:
  • Exp(+Inf) = +Inf
  • Exp(NaN) = NaN

    • Very large values overflow to 0 or +Inf.
    Very small values underflow to 1.

    func Exp2(x)
    Exp2 returns 2**x, the base-2 exponential of x.
    Special cases are the same as Exp.

    func Ldexp(frac, exp)
    Ldexp is the inverse of Frexp. It returns frac × 2**exp.
    Special cases are:
  • Ldexp(±0, exp) = ±0
  • Ldexp(±Inf, exp) = ±Inf
  • Ldexp(NaN, exp) = NaN


    • func Expm1(x)
    Expm1 returns e**x - 1, the base-e exponential of x minus 1.
    It is more accurate than Exp(x) - 1 when x is near zero.
    Special cases are:
  • Expm1(+Inf) = +Inf
  • Expm1(-Inf) = -1
  • Expm1(NaN) = NaN

    • Very large values overflow to -1 or +Inf.

    • func FMA(x, y, z)
    FMA returns x * y + z, computed with only one rounding.
    (That is, FMA returns the fused multiply-add of x, y, and z.)

    func Frexp(x)
    Frexp breaks f into a normalized fraction and an integral power of two.
    It returns frac and exp satisfying f == frac × 2**exp, with the absolute value of frac in the interval [½, 1).
    Special cases are:
  • Frexp(±0) = ±0, 0
  • Frexp(±Inf) = ±Inf, 0
  • Frexp(NaN) = NaN, 0


    • func Gamma(x)
    Gamma returns the Gamma function of x.
    Special cases are:
  • Gamma(+Inf) = +Inf
  • Gamma(+0) = +Inf
  • Gamma(-0) = -Inf
  • Gamma(x) = NaN for integer x < 0
  • Gamma(-Inf) = NaN
  • Gamma(NaN) = NaN


    • func Hypot(p, q)
    Hypot returns Sqrt(p*p + q*q), taking care to avoid unnecessary overflow and underflow.
    Special cases are:
  • Hypot(±Inf, q) = +Inf
  • Hypot(p, ±Inf) = +Inf
  • Hypot(NaN, q) = NaN
  • Hypot(p, NaN) = NaN

    • func Log(x)
    Log returns the natural logarithm of x.

    • func Log10(x)
    Log10 returns the decimal logarithm of x.
    The special cases are the same as for Log.

    func Log2(x)
    Log2 returns the binary logarithm of x.
    The special cases are the same as for Log.

    func Log1p(x)
    Log1p returns the natural logarithm of 1 plus its argument x.
    It is more accurate than Log(1 + x) when x is near zero.
    Special cases are:
  • Log1p(+Inf) = +Inf
  • Log1p(±0) = ±0
  • Log1p(-1) = -Inf
  • Log1p(x < -1) = NaN
  • Log1p(NaN) = NaN


    • func Lgamma(x)
    Lgamma returns the natural logarithm and sign (-1 or +1) of Gamma(x).
    Special cases are:
  • Lgamma(+Inf) = +Inf
  • Lgamma(0) = +Inf
  • Lgamma(-integer) = +Inf
  • Lgamma(-Inf) = -Inf
  • Lgamma(NaN) = NaN


    • func Logb(x)

    Special cases are:
  • Logb(±Inf) = +Inf
  • Logb(0) = -Inf
  • Logb(NaN) = NaN


    • func Ilogb(x)
    Ilogb returns the binary exponent of x as an integer.
    Special cases are:
  • Ilogb(±Inf) = MaxInt32
  • Ilogb(0) = MinInt32
  • Ilogb(NaN) = MaxInt32


    • func Modf(x)
    Modf returns integer and fractional floating-point numbers that sum to f. Both values have the same sign as f.
    Special cases are:
  • Modf(±Inf) = ±Inf, NaN
  • Modf(NaN) = NaN, NaN


    • func Nextafter(x, y)
    Nextafter returns the next representable float64 value after x towards y.
    Special cases are:
  • Nextafter(x, x) = x
  • Nextafter(NaN, y) = NaN
  • Nextafter(x, NaN) = NaN


    • func Pow10(n)
    Pow10 returns 10**n, the base-10 exponential of n.
    Special cases are:
  • Pow10(n) = 0 for n < -323
  • Pow10(n) = +Inf for n > 308

    • func Floatbits(f)
    Floatbits returns the IEEE 754 binary representation of f, with the sign bit of f and the result in the same bit position, and Floatbits(Floatfrombits(x)) == x.

    func Floatfrombits(b)
    Floatfrombits returns the floating-point number corresponding to the IEEE 754 binary representation b, with the sign bit of b and the result in the same bit position.
    Floatfrombits(Floatbits(x)) == x.

    func Y0(x)
    Y0 returns the order-zero Bessel function of the second kind.
    Special cases are:
  • Y0(+Inf) = 0
  • Y0(0) = -Inf
  • Y0(x < 0) = NaN
  • Y0(NaN) = NaN


    • func Y1(x)
    Y1 returns the order-one Bessel function of the second kind.
    Special cases are:
  • Y1(+Inf) = 0
  • Y1(0) = -Inf
  • Y1(x < 0) = NaN
  • Y1(NaN) = NaN


    • func Yn(n, x)
    Yn returns the order-n Bessel function of the second kind.
    Special cases are:
  • Yn(n, +Inf) = 0
  • Yn(n ≥ 0, 0) = -Inf
  • Yn(n < 0, 0) = +Inf if n is odd, -Inf if n is even
  • Yn(n, x < 0) = NaN
  • Yn(n, NaN) = NaN
