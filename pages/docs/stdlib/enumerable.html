enumerable

Functions

func Map(var ienum, var fun)
Map returns results after applying the given function to each item of a given iterable object.
Special cases are:
  • Map(ienum, fun) = error if "ienum" is not list, map or string
  • Map(ienum, fun) = error if "fun" is not function

    • Panic cases are:
  • Panics if function is not compatible.
  • Function is not: func(accessor, element) = returns value for set as element
  • Panics if returned value of function is not compatible with iterable object.

    • Classes

    StrongList

    StrongList includes only elements compatible with specified type code.

    Constructor

    func StrongList(typeCode, ...elements=[])
    Special case is;
  • StrongList(typeCode) = panics if typeCode is not integer.

    • Indexer

    func StrongList[index]
    Indexer returns element as constant by specified index. Supports reverse indexing.
    Special cases are:
  • StrongList[index] = panics if index is not integer.
  • StrongList[index] = panics if index out of bounds.

    func PushFront(...elements)

    Panic cases are:
  • PushFront(elements) = panics if any element is not compatible strong type of list.

  • Insert(index, elements) = panics if index out of bounds.

    Removes element at specified index.

    • func Remove(element, start=0)
    Remove first matched element. Returns true value is removed, false if not. Start parameter provides start position of search.
    Special cases are:
  • Remove(element) = false if element is not compatible with strong type of list.
  • Remove(element, start) = panics if index is not integer.
  • Remove(element, start) = panics if index out of bounds.


    • func Set(index, element)

  • Set(index, element) = panics if index is not integer.
  • Set(index, element) = panics if index out of bounds.


    • func RemoveAll(element)
    Removes all matched values in buffer.
    Special case is are:
  • RemoveAll(element) = nothing if any element is not compatible with strong type of list.


    • func Index(element, start=0)
    Returns index of first matched element if any matched element, -1 if not. Start parameter provides start position of search.
    Special cases are:
  • Index(element) = -1 if element is not compatible with strong type of list.
  • Index(element, start) = panics if start is not integer
  • Index(element, start) = panics if start out of bounds
