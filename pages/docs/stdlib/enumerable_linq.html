func Max(lst, filter=none)
Max returns maximum value in list by filter.
Special cases are:
  • Max(lst) = none if lst is not list
  • Max(lst) = none if lst is empty
  • Max(lst, none) = Not filter elements

    • Panics if filter function is not compatible.
    Filter function: func(x) -> bool
