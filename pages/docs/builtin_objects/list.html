    • Functions

    func pushFront(...v)
    Append values to start of list.

    func pushBack(...v)
    Append values to end of list.

    func insert(i, ...v)
    Append values to list start at position.
    Special cases are:
  • insert(i, v) = panics if i is not integer
  • insert(i, v) = panics if index out of bounds


    • func removeAt(i)
    Remove element by index.
    Special cases are:
  • removeAt(i) = panics if i is not integer
  • removeAt(i) = panics if index out of bounds


    • func remove(v, start=0)
    Remove first matched element. Returns true value is removed, false if not. Start parameter provides start position of search.
    Special cases are:
  • removeLast(v, start) = panics if start is not integer
  • removeLast(v, start) = panics if start out of bounds


    • func removeLast(v, start=$len-1)
    This function is same with remove function. Only difference is count down from list.
    Special cases are:
  • removeLast(v, start) = panics if start is not integer
  • removeLast(v, start) = panics if start out of bounds


    • func removeAll(v)
    Remove all matched elements.

    func removeRange(start, to)
    Remove elements between at start index and to specified end index.
    Special cases are:
  • removeRange(start, end) = panics if start is not integer
  • removeRange(start, end) = panics if start out of bounds
  • removeRange(start, end) = panics if end is not integer
  • removeRange(start, end) = panics if end out of bounds


    • func index(v, start=0)
    Returns index of first matched element if any matched element, -1 if not. Start parameter provides start position of search.
    Special cases are:
  • index(v, start) = panics if start is not integer
  • index(v, start) = panics if start out of bounds


    • func indexLast(v, start=$len-1)
    This function is same with index function. Only difference is count down from list.
    Special cases are:
  • indexLast(v, start) = panics if start is not integer
  • indexLast(v, start) = panics if start out of bounds


    • func include(v, start=0)
    Returns true if any matched element, false if not. Start parameter provides start position of search.
    Special cases are:
  • include(v, start) = panics if start is not integer
  • include(v, start) = panics if start out of bounds


    • func clear()
    Remove all elements.

    func reverse()
    Reverse elements.

    func sort(desc=false)
    Sort elements.
    Special case is are:
  • sort(desc) = panics if dest is not boolean


    • func unique()
    Returns each element maximum one repetition in another list.
