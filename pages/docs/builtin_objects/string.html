    Functions

    fn isLower()
    Returns true if all chars is lowercase, false if not.

    fn isUpper()
    Returns true if all chars is uppercase, false if not.

    fn lower()
    Parse all characters to lowercase and returns in new string.

    fn upper()
    Parse all characters to uppercase and returns in new string.

    fn trim()
    Remove all space characters and returns in new string.

    fn trimLeft()
    Remove all space at left of string characters and returns in new string.

    fn trimRight()
    Remove all space at right of string characters and returns in new string.

    fn sub(start, len)
    Select charaters start at specified start index by specified selection length and returns as string.

    fn index(sub)
    Returns index of the first instance of sub in string, or -1 if sub is not present in string. Start parameter provides start position of search.

    fn indexLast(sub)
    Returns index of the last instance of sub in string, or -1 if sub is not present in string. Start parameter provides start position of search.

    fn split(sep, count=1)
    String into substrings separated by sep and returns a list of the substrings between those separators. If count less than 0, means unlimited.

    fn hasPrefix(sub)
    Returns true string is startss with specified substring, false if not.

    fn hasSuffix(sub)
    Returns true string is ends with specified substring, false if not.

    fn replace(old, new, count=1)
    Replace old strings with specified new string by specified replacement count.

    fn replaceAll(old, new)
    Replace all matched old strings with specified new string.
